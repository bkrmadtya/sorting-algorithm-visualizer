### Tech stack:

- React
- TypeScript
- Redux
- React-Router
- SASS
- Babel
- Webpack
- Eslint

### To run the project locally, clone the project and run:

```bash
npm install && npm start
```

Or

```bash
yarn install && yarn start
```

Or

```bash
pnpm install && pnpm start
```
